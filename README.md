# ROS RaspberryPi Robot/AGV  

Lidar guided autonomous robot.  


## Hardware  

* Raspberry Pi 3  
* L298N dual H-Bridge  
* Ydlidar X4  
* LiPo 3.7V JST PH 2.0

## Setup  

* Ubuntu 18.04 LTS
* ROS Melodic  


## Start  

```
* catkin_make  
* source /devel/setup.bash  
*  roslaunch raspi_ctrl raspi.launch  
```
